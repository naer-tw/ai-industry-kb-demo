DeepSeek 模型

一句話定義

DeepSeek（深度求索）是中國杭州由量化基金 High-Flyer（幻方量化）孵化的 AI 實驗室，以「高性價比、開源權重（MIT）的 Mixture-of-Experts 大模型與推理模型」聞名；其 R1 推理模型於 2025 年 1 月引爆全球「DeepSeek 衝擊」，並重塑了開源/低成本訓練的論辯。

  • DeepSeek-V3：2024-12 發布，671B 總參數 / 每 token 啟動 37B（MoE），MIT 授權。MMLU 88.5、MMLU-Pro 75.9、GPQA 59.1（V3 技術報告 arXiv:2412.19437）。[來源4]

在 AI 供應鏈的位置與角色

  • 屬中游「模型層」：自行訓練基礎模型與推理模型，並以 API + 開源權重雙軌對外。
  • 對上游晶片（NVIDIA）：宣稱以受出口管制的 H800 等「次級」晶片做到高效訓練，是「算力效率」敘事核心；但第三方（SemiAnalysis）估其累計硬體投入逾 $1.6B、約 5 萬張 GPU。[來源7][來源8 旁證]
  • 對下游應用：MIT 授權＋極低 API 價，使其成為新創、開源社群與中國雲廠（華為、寒武紀已適配 V4）最常用的低成本替代方案。[來源1]
  • 競爭定位：以「對標 OpenAI o1 / GPT 系列、價格僅 1/10」的性價比切入，是西方閉源旗艦的主要開源挑戰者之一。
