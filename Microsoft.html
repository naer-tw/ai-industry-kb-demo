AI產業知識庫公司

Microsoft

一句話定義

微軟是全球最大軟體與雲端公司之一，透過 Azure 雲端、Microsoft 365 Copilot 與對 OpenAI 的巨額持股，成為 AI 供應鏈「中游」的核心算力平台與企業 AI 應用分發者；同時自 2025 年起以 MAI 自研模型團隊向上游基礎模型延伸。

  • 資本支出（capex）FY2025 達 646 億美元，較前一年增約 50%，主要投入 AI 資料中心。(來源1, 2025-07)
  • Q2 FY2026 營收 813 億美元（年增 17%），營業利益 383 億美元（年增 21%），Azure 及其他雲端服務年增 39%（固定匯率 +38%）；Microsoft Cloud 單季突破 500 億美元。(來源2／來源7, 季度截至 2025-12-31)

在 AI 供應鏈的位置與角色

微軟主要位於 AI 供應鏈「中游（雲端／平台層）」，並橫跨多個環節： - 算力平台（中游核心）：Azure 提供 GPU 雲端訓練與推論，是 OpenAI GPT 系列、Anthropic Claude 系列 等前沿模型的關鍵部署平台；Azure AI Foundry（原 Azure AI / Foundry）為企業多模型目錄，2025 年起成為唯一同時提供 GPT 與 Claude 前沿模型的雲端。(來源6) - 應用分發（下游介面）：Microsoft 365 Copilot、GitHub Copilot、Copilot Studio 將 AI 嵌入 Office、開發工具與企業流程，是 AI Agent 商業化（AI Agent與自動化）最大規模的企業通路之一。 - 基礎模型（上游延伸）：2025 年成立 MAI Superintelligence Team，推出 MAI 系列自研模型（含推理模型 MAI-Thinking-1），由依賴 OpenAI 轉向「自研 IP」雙軌。(來源5) - 晶片採購方：大量採購 NVIDIA GPU，並導入 AMD MI 系列與自研 Maia/Cobalt 晶片（部分由 Broadcom 等協力），是上游硬體最大買家之一。

現況與最新動態（2024–2026）

  • 2025-10-28 OpenAI 重組與新協議：OpenAI 轉型為 OpenAI Group PBC，微軟持股約 27%（約 1,350 億美元）；OpenAI 承諾再採購 2,500 億美元 Azure 服務；微軟放棄對 OpenAI 雲端採購的優先承購權；OpenAI 在達成 AGI（須經獨立專家小組驗證）前持續分潤 20% 給微軟。(來源3／來源4, 2025-10-28)
  • 2025-11-06 MAI 超級智慧團隊：Mustafa Suleyman 宣布成立「MAI Superintelligence Team」，提出「人本超級智慧（Humanist Superintelligence, HSI）」願景；新協議解除對微軟自研模型規模上限的限制。(來源5, 2025-11)
  • 2025-11 Anthropic／Nvidia 三方合作：微軟對 Anthropic 投資約 50 億美元、NVIDIA 至多 100 億美元；Anthropic 承諾採購約 300 億美元 Azure 算力；Claude 系列（Sonnet 4.5、Opus 4.x、Haiku 4.5）進駐 Microsoft Foundry 與 Microsoft 365 Copilot。(來源6, 2025-11)
  • Q2 FY2026（2026-01-28 公布）：Microsoft Cloud 單季破 500 億美元，Azure 年增 39%；但 Azure 連兩季略低於部分市場高標預期，並揭露因電力／資料中心產能限制而無法立即履約的 Azure 訂單積壓（backlog）達約 800 億美元，顯示需求超前建置速度。(來源2／來源7, 2026-01)

危機 / 變化 / 風險

  • 與 OpenAI 關係從「準獨家」轉為「競合並存」：新協議放寬獨家綁定，微軟同時投資 Anthropic 並自研 MAI，可能與最大合作夥伴 OpenAI 在前沿模型與企業客戶上正面競爭。(來源3／來源5)
  • 資本支出與獲利壓力：年化 capex 逼近千億美元級距，市場對 AI 投資回報週期、折舊侵蝕利潤率高度關注；Azure 連兩季未達高標預期已引發估值討論。(來源7, 2026-01)
  • 算力供給瓶頸：約 800 億美元 Azure 訂單因電力與資料中心產能受限無法即時履約，建置速度成為成長天花板。(來源7, 2026-01)
  • Copilot 滲透率仍偏低：1,610 萬付費席次相對 4.5 億以上 M365 商用付費座位（截至 2025-12-31），付費滲透率僅約 3.6%（部分分析估約 3.3%），存在「買了未用（shelfware）」與變現節奏質疑。(來源9／office365itpros, 2026-01)
  • 監管風險：對 OpenAI 持股與多家模型公司投資，持續面臨美、歐反壟斷與資料中心能耗、競爭審查關注（OpenAI 重組已獲德拉瓦州檢察長等核可，但監管目光未散）。(來源4, 2025-10)
