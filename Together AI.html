Together AI

一句話定義

Together AI（公司原名 Together Computer Inc.）是專注於「開源／開放權重模型」的 AI 雲端推論與訓練平台，一邊以 per-token API 提供 Llama、DeepSeek、Mistral 等開源模型的即時推論，一邊出租 NVIDIA GPU 叢集供企業訓練與微調，被視為夾在原始 GPU 雲（如 CoreWeave）與超大規模雲之間、押注「開源模型推論」這一賽道的中游代表業者。

在 AI 供應鏈的位置與角色

Together AI 位居 AI 供應鏈的中游（模型服務／推論雲層），扮演「開源模型 × 算力」的整合分發者： - 向上游採購：大量使用 NVIDIA GPU（Hopper、Blackwell），部分算力自建資料中心、部分向 CoreWeave、Lambda 等 GPU 雲與學術機構租用，因此既是 NVIDIA 的下游買家，也與 NVIDIA 有股權綁定（NVIDIA 為其投資人）。 - 承載開放模型：平台託管 100＋ 開源／開放權重模型，包括 Meta 的 Llama、Mistral AI、DeepSeek 等，使其成為「非 OpenAI / Anthropic 封閉 API」之外的重要替代推論管道。 - 服務下游：對應用開發者與企業提供 API 推論、微調、專屬叢集，與 Hugging Face、Fireworks、DeepInfra、OpenRouter 等推論服務商，以及 Groq、Perplexity 等在「推論速度／成本」維度競爭。 - 與封閉模型陣營的關係：作為開源推論平台，Together AI 在策略上與 OpenAI、Anthropic 的封閉 API 形成替代與互補並存——企業可用其部署開源模型以降低成本、避免單一供應商鎖定。
