① 基礎模型實驗室

前沿模型由少數重資本實驗室主導，分屬「閉源」與「開源權重」兩大陣營：

  • OpenAI：GPT 系列開發商，與 Microsoft 深度綁定（持股約 27%）；2025 年 ARR 約 200 億美元，ChatGPT 週活約 9 億。
  • Anthropic：Claude 系列母公司，企業級 LLM 市占龍頭；H 輪後投後估值 9,650 億美元，年化營收 2026-05 突破 470 億美元（來源見 Claude 系列）。主要算力夥伴為 Amazon 與 Google。
  • Google DeepMind：Gemini 開發單位，少數自研晶片（TPU）的垂直整合者，不依賴 Microsoft 等外部雲。
  • Meta：Llama 開源權重旗手；2026-04 推出閉源 Muse Spark，轉為「開源 Llama 5 ＋閉源 Muse Spark」雙軌。
  • xAI：Grok 開發商，靠 Colossus 超大算力叢集硬拚 SOTA；2026-02 併入 SpaceX。
  • Mistral AI：歐洲主權 AI 代表，開放權重路線，ASML 為大股東。
  • DeepSeek：中國高性價比開源（MIT）實驗室，2025-01 R1 引爆全球「DeepSeek 衝擊」。

③ 訓練／推論框架與技術

模型能力的底層，是一整套架構與軟體堆疊：

  • 架構層：Transformer（2017《Attention Is All You Need》，當今幾乎所有 LLM 的共通底座）與 MoE混合專家（稀疏啟動，使總參數與推論成本脫鉤；DeepSeek 模型、Llama 系列、Qwen 系列 皆採用）。
  • 軟體堆疊層：CUDA（NVIDIA 專有平行運算平台，被視為最持久的「軟體護城河」，累積約 600 萬開發者）與 PyTorch（事實標準深度學習框架，由 Meta 創建、現屬 Linux Foundation）。
  • 服務層：推論優化（PagedAttention／vLLM、KV cache 量化、推測解碼等，把昂貴算力轉成可負擔的 token 單價）與 RAG檢索增強生成（外部知識接地、降低幻覺，市場估 2030 年達 98.6 億美元）。

④ 雲端平台分發

旗艦模型最終透過超大規模雲端與各家 API 觸及客戶：

  • Microsoft：Azure ＋ Microsoft Foundry，唯一同時提供 GPT 與 Claude 前沿模型的雲；365 Copilot 為最大企業通路之一。
  • Amazon：AWS ＋ Amazon Bedrock 託管 Claude 系列 等模型，並以自研 Trainium／Inferentia 與對 Anthropic 鉅額投資綁定旗艦。
  • Oracle：OCI 強勢切入算力租賃，與 OpenAI 簽 3,000 億美元 Stargate 雲端合約，RPO 暴增至 5,530 億美元。
  • 多雲分發常態：Claude 系列 同時上架 AWS Bedrock、Vertex AI（Google）、Microsoft Foundry；Gemini 系列 走 Vertex AI；開源權重模型則經 Hugging Face、Together、Fireworks、Groq 等託管分發。

風險與張力

  • 領先脆弱：前沿排名每數週易主，閉源三強差距常僅個位數分。
  • 算力依賴：訓練／推論高度仰賴上游 NVIDIA GPU、HBM 與資料中心電力，受先進製程與能源瓶頸牽動。
  • 成本與定價：旗艦閉源（如 Opus、GPT-5.5）維持高價位，面臨開源與對手降價雙重壓力。
  • 地緣與智財：中國模型（Qwen 系列、DeepSeek 模型）面臨出口管制與採購限制；Anthropic、OpenAI 指控對手「蒸餾」前沿模型，牽動 AI 智財邊界。
  • 估值與泡沫：實驗室估值與資本支出狂飆，與「AI 泡沫」疑慮並存。
