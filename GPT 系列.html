GPT 系列

一句話定義

GPT 系列是 OpenAI 的旗艦大型語言模型家族，自 GPT-1（2018）演進至 GPT-5.x（2025–2026）的統一推理多模態系統，是當前生成式 AI 供應鏈中游「基礎模型」的標竿產品線之一。

關鍵事實與數據（每項附來源與時間）

  • 版本時間線（2025–2026）：GPT-5 於 2025-08-07 發布，為首個「統一系統」（自動決定快答或長思考）（OpenAI，2025-08-07）。其後迭代：GPT-5.1（2025-11-13）、GPT-5.2「Garlic」（2025-12-10/11）、GPT-5.5（2026-04-23）、GPT-5.5 Instant（成為 ChatGPT 預設，2026-05-05）（OpenAI 官方公告／TechCrunch，2025–2026）。
  • GPT-5 原版 benchmark（OpenAI，2025-08）：AIME 2025 達 94.6%（無工具）、SWE-bench Verified 74.9%、Aider Polyglot 88%、MMMU 84.2%、GPQA 88.4%（Pro 延伸推理，無工具）、HealthBench Hard 46.2%。
  • GPT-5.5 benchmark（O-mega.ai／Converge AI 整理，2026-04）：MMLU 92.4%、GPQA Diamond 93.6%、SWE-bench Verified 88.7%、SWE-bench Pro 58.6%、Terminal-Bench 2.0 82.7%、OSWorld-Verified 78.7%、ARC-AGI-2 85.0%、MRCR v2（512K–1M）74.0%。註：此為第三方彙整，部分數字尚待 OpenAI 模型卡逐項對照證實。
  • 上下文窗口：GPT-5.2 為 400K token 輸入／128K 輸出（eWeek／OpenRouter，2025-12）；GPT-5.5 API 擴至 1,000,000 token 輸入／128K 輸出（Codex 環境實測約 258K–400K）（第三方整理＋openai/codex GitHub issue #19319，2026）。
  • 定價（每百萬 token）：GPT-5 原版輸入 $1.25／輸出 $10.00（2025-08）；GPT-5.2 輸入 $1.75／輸出 $14.00（2025-12）；GPT-5.5 輸入 $5.00／輸出 $30.00，較 5.x 線約翻倍（pricepertoken／apidog，2026-04）。
  • 開放權重支線：gpt-oss-120b 與 gpt-oss-20b 於 2025-08-05 以 Apache 2.0 釋出，MoE 架構（120b 每 token 啟用 5.1B、20b 啟用 3.6B），可在單張 80GB GPU 或 16GB 邊緣裝置執行；120b 接近 o4-mini、20b 接近 o3-mini（OpenAI，2025-08-05）。

在 AI 供應鏈的位置與角色

  • 中游「基礎模型層」：GPT 系列是上游算力（NVIDIA GPU、微軟 Azure 等）與下游應用（ChatGPT、API 開發者、企業整合）之間的核心引擎。
  • OpenAI 透過 API 與 ChatGPT 雙軌變現：API 服務開發者與企業，ChatGPT 面向消費者（GPT-5.5 Instant 為預設模型）。
  • 與微軟深度綁定（算力、通路、投資），同時是 Anthropic（Claude 系列）、Google（Gemini 系列）的主要競爭者，三者在 Artificial Analysis 智慧指數上差距僅約 3 分（2026-04）。

現況與最新動態（2024–2026）

  • 主力為 GPT-5.5（2026-04-23），官方稱為「最聰明、最直覺」的模型，並首次是自 GPT-4.5 以來的完整重新預訓練（第三方整理）。
  • GPT-5.5 Instant 自 2026-05-05 成為 ChatGPT 預設，官方稱在醫療／法律／金融等高風險題上幻覺主張較 GPT-5.3 Instant 減少 52.5%（OpenAI／TechCrunch，2026-05）。
  • 在 agentic（自主代理）領域，GPT-5.5 於 Terminal-Bench（82.7%）、OSWorld（78.7%）領先 Claude Opus 4.7，被視為 OpenAI 自 GPT-4 以來首次在代理執行上反超 Anthropic（第三方測評，2026-04）。
  • 傳聞 GPT-5.6 預計 2026-06 發布，主打進階推理與 agentic 工作流（Geeky-Gadgets，2026；屬媒體預測，資料未由官方證實）。

危機 / 變化 / 風險

  • 競爭壓力「Code Red」：2025-12 Google Gemini 3 在多項 benchmark 超車，Sam Altman 對內發布「code red」備忘錄、暫停其他專案集中火力，催生 GPT-5.2（Fortune／Tom’s Hardware／CNBC，2025-12）。企業市佔報導顯示 OpenAI 約 27%、Gemini 約 21%、Anthropic 約 40%（Fortune，2025-12，數據隨來源而異）。
  • 預訓練邊際遞減：有報導指 OpenAI 2025 年多次大型預訓練未能顯著超越前代，為內部焦慮來源（Fortune，2025-12，屬報導未經官方證實）。
  • 幻覺爭議：Artificial Analysis 指 GPT-5.5 在 AA-Omniscience（純靠自身權重、無檢索）幻覺率約 86%，為 Claude Opus 4.7 兩倍以上；顯示無 grounding 時可靠度仍是弱點（usewire／digitalapplied，2026）。
  • 漲價風險：GPT-5.5 較前代約翻倍定價，對高用量開發者成本壓力上升（apidog／pricepertoken，2026-04）。
